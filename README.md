# ToDoList-forYT
 可插入各網頁的代辦事項及瀏覽時間統計與限制
###### *version-V2.0* 
---
## 簡介&功能

1. 統計瀏覽時間
    - 點選擴充圖標查看 
2. 設定分類標籤
    - 右鍵點選圖標進入選項頁面->**規則設定**
3. 限制瀏覽時間
    - 右鍵點選圖標進入選項頁面->**封鎖**（**Advance notice**以及**星期幾設定**未實現）
4. 待辦事項頁面
    - 點選擴充圖標->**Todo**（左下角）
## 下載&安裝
1.下載資料夾並解壓縮：code -> download ZIP 

2.開啟擴充頁面並打開開發者模式：擴充功能圖示(右上) -> 管理擴充功能 -> 開發人員模式

3.載入擃充資料夾：載入未封裝項目(載入解壓縮) -> 選擇下載的資料夾
